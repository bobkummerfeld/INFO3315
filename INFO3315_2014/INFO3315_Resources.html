Text and reference books

Textbook
Rex Hartson and Pardha S Pyla,
The UX Book: Process and Guidelines for Ensuring a Quality User Experience,
Elsevier, 2012.
In the lists of reading materials, this will be fondly referred to as H&P.
Reference Books for the UX aspects

The first is a classic, widely used introductory HCI text.
It is available from the SciTech Library.
Yvonne Rogers, Helen Sharp and Jenny Preece,
Interaction Design: beyond human-computer interaction
3rd Edition, 2011.

The next is an excellent older book with a stronger software engineering focus.
It is out of print and not available in the library.
William M. Newman, Michael G. Lamming, and Mik Lamming. 1995.
Interactive System Design.
Addison-Wesley Longman Publishing Co., Inc., Boston, MA, USA.

The next two are more advanced, and valuable for conducting rigorous user studies.
Both are available online from the library at the links below:

MacKenzie, I. Scott.
Human-Computer Interaction, 1st Edition An Empirical Research Perspective,
Elsevier, 2013.

Albert, Bill (William)
Measuring the User Experience:Collecting, Analyzing, and Presenting Usability Metrics,
Morgan Kaufman, Elsevier, 2008.

Online resources for technical aspects
There is a huge number of resources for learning web tech.
This list is based on resources that many have found useful in the past.

  • Ground up Mercurial A friendly introduction to the Mercurial DVCS by Joel Spolsky. This introduces command line hg.
  • Quickstart Mercurial is a slightly more detailed guide.
  • A Tutorial on Mercurial is longer.
  • The W3Schools website has many tutorials and reference guides on web technology.
  • W3Schools CSS tutorial
  • CSS reference material.
  • Dash also has an excellent tutorial on CSS.
