  The Document Object Model - DOM (Homework)

Study the Javascript HTML DOM tutorial up to the end of "DOM Events".

As a simple exercise, write a javascript function that allows the user to display or hide a block of text when a button is clicked.

You can put you text in a "div" block and give it an id. 

 <div id="mytext"> This text will be hidden </div>
Then you can have an image that, when clicked, will hide or reveal your text. You can do this by associating a javascript function with the image. 
 <img src="url of image file" onclick="javascript: showhide('mytext');">
Put the "showhide" javascript function between "script" tags in the head section of your html file.

  Combining server, DOM and AJAX (Lab)

As an exercise use AJAX with your web page to load some html text. Use a "bottle" server to handle the request.

To do this, first create an empty "div" on your page and give it an id. Then, as in the DOM exercise, create a clickable image that invokes a javascript function. The function should use xmlhttp as shown in the tutorial to fetch text from a URL. Make the URL refer to your bottle server, eg http://127.0.0.1:54321/get_text.

In your bottle server you have a function that will be called when the AJAX URL is used and have it send back some HTML text. As you saw in the module on bottle you can do this easily with a function decorator "@route('/get_text')".

Back in your web page you can take the result of the xmlhttp request and display by storing it in the empty div using the DOM.

Start by using the synchronous form of AJAX and then modify it to use an asynchronous call.
