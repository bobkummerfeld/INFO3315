  Cascading Style Sheets (Homework)

This is a quick overview of CSSS and its interaction with HTML, as a taster for the tutorial linked at the bottom of this cell.

Last week we saw how the HTML markup language can be used to describe the content of a web page. The original aim of the HTML design was simply to describe the content and leave the styling and layout to some other part of the system (eg the browser). Unfortunately, HTML was "enhanced" with many tags that specified style and layout. For example, the "<b>" tag was introduced to specify that the text was to be rendered in bold. More recently, the HTML language has been revised and another language (CSS) introduced to specify the style and layout of elements on the page.

The CSS language can be written inside your HTML file or it can be written in a separate file and loaded when your web page is loaded.
Small snippets of CSS can even be included with individual tags in the HTML.
For this tutorial, we suggest that you put all the CSS statements at the beginning of your html body section between <style> and </style> tags.

CSS consists of a set of rules that say how HTML elements whould rendered (arranged on the page and styled).
A CSS rule consists of a selector and a declaration or block of declarations.
The selector is used to select an individual HTML element or set of HTML elements in your page.
Here is an example of changing the style of paragraphs in the document.
You add the CSS statement: 

p { font-size:120%; }

Try adding this CSS and the style tags to the homepage you already made.

This would make the paragraph text large, as in this line.

In this case the selector is the "p" tag (paragraph) and the block contains one declaration setting the font size for all text enclosed in the p tag to 120% of the original size.

We can add more declarations inside the block: 

p { font-size:120%; color:red;}

Try altering your homepage, to have this additional declaration.
It should make the paragraph text look like this:

This now makes the paragraph text large and red.

This will define the appearance of all paragraphs.
However, you may not want all paragraphs in this new style.
CSS has several mechanisms for selecting individual paragraphs or classes of paragraph.

We now show how to do this.
HTML tags can have attributes.
These are name/value pairs that are included in the angle brackets.

For example, suppose we have set up the style above but we now want to be able to have some paragraphs that are even bigger and have the text in blue.
We can give the p selector the id "biggerblue", as in this line of HTML.
We show it with a grey background to distinguish it from the CSS examples above: 

<p id=biggerblue>This paragraph will be bigger and blue.</p>

if we provide the following CSS statement (in the style section) defining the font and colour of this <p> tag: 

#biggerblue { font-size:150%; color:blue;}

and then the paragraph will appear bigger and blue.
Try it!
Add the extra CSS line and the HTML line as above, each in the right part of your file. You will like it!
Notice the "#" character.
This indicates that we are specifying an id rather than a tag name.
It is used to specify an individual tag.

You can also have classes of tag:


<p class=smallgreen>This paragraph will be small and green.</p>

where we have the selector: 

.smallgreen { font-size:60%; color:green;}

("." character leading the class name)
All tags with that class will then have the style applied.

If you have multiple tags, ids or classes with the same declarations you can combine them. For example: 

p table { font-size:120%; color:red;}

This will apply the large red font to both paragraphs and tables.

The basic idea of CSS is that you can select an entity (eg text, table, paragraph etc) and say how it should look.

The other principle of HTML/CSS is that all the entities that are to be displayed are considered to be "boxes".
Every box has standard attributes. Some basic ones are margins, borders, padding, background and content that can be modified.
For example, borders can be transparent or solid or dashed or have many other special effects applied.

A paragraph in a dotted border box is styled: 

p { border-style:dotted; }

and appears

like this

To recap, the main principles for HTML/CSS are:

  • HTML and CSS are separate languages: HTML specifies the content, CSS specifies the style of the content
  • HTML and CSS can be in the same file, different files or in-line
  • CSS rules consist of selectors to select one or more tags, and declarations (or rules/properties) to say how they should be styled
  • Entities on the page are built from boxes with changeable characteristics.

For a detailed study of CSS attributes and how to use them, please refer to the online resources at W3Schools.
You should now work steadily through the excellent W3Schools tutorial.
Also, skim the attributes listed on the reference material.
Get a sense of what they offer.
The Dash web site also has an excellent tutorial on CSS.
