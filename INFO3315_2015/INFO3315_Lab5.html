Introduction to Javascript and Web Toolkits (Homework)

The combination of HTML and CSS allows us to create web content with a range of fonts, styles, colours and layout. This can be enhanced even further with the addition of the third major web technology, Javascript. Javascript is a programming language that can be added to web pages to provide dynamic behaviour, eg an action occurs when you click on an element of the page.

This week we will introduce Javascript and also the jQuery and Bootstrap toolkits that provide even more powerful services using a combination of Javascript and CSS.

  Javascript (Homework)

Javascript is one of three technologies that make up the web: HTML, CSS and Javascript. It is essential for web developers to learn Javascript.

Javascript is a general purpose programming language that can be embedded in html to respond to user actions (eg key presses, mouse clicks). It can also be used as a stand alone programming outside the browser and is gaining popularity for "server-side" applications.

Please note that Javascript has nothing to do with the Java programming language apart from some common syntax elements that were inherited from other languages such as C. The formal name for the version of javascript in use today is "ECMAscript" after the European standards organisation that has created the formal standard for the language.

The javascript language has dynamic typing and supports object oriented, functional and imperative programming styles. See the wikipedia article for more detail.

In a similar way to CSS we can embed javascript statements in the <head> or <body> sections of the html file. We can also link to an external javascript file with the "src=" attribute in a <script> tag.

Now please complete the W3Schools Javascript Tutorial. Do each part up to at least the section on Javascript Strings.

  Bootstrap (Homework)

Bootstrap is a very popular HTML, CSS and Javascript framework that allows you to easily build responsive web sites that adapt to different screen sizes and systems, including mobile.

Bootstrap includes a number of "themes" that will give your website a look and feel that is popular on web sites at present (2014). Take a look at the examples on the bootstrap web site.

There are many online tutorials for Bootstrap. We suggest you do this one.

  jQuery (Homework)

Javascript is a powerful language for making your web pages dynamic and responsive but there are many features that you may want to add to your web page that required a lot of javascript code. Also, these features a often generally useful so it makes sense to use a pre-written library of javascript functions that provide the feature.

There are many javascript libraries available, but the most popular is jQuery, used by the majority of web sites. You can read the background and features of jQuery in the wikipedia article.

Please do the W3schools jQuery tutorial. Do each part up to the jQuery Animate section.

There are a number of ways of including the jQuery library in your web page. We recommend using the version hosted by google by placing the following code in the <head> section of your html: 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
